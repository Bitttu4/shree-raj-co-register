# SHREE RAJ & CO Register Management System

## Frontend

- Expo SDK 56
- React Native
- TypeScript
- Expo Router

## Backend

- FastAPI
- MongoDB (Motor)
- JWT Authentication

## Features

- Dashboard
- Client Management
- Document Management
- Task Management
- CSV Import

## Authentication

- JWT based login/signup
- AuthContext
- SecureStore

## Database

Collections:

- users
- clients
- documents
- tasks

## Current Architecture

Frontend → REST API → FastAPI → MongoDB

## Goal

Convert the application into:

- Single-user app
- Completely offline
- No backend
- No MongoDB
- No login/signup
- Local data only
- Use expo-sqlite
- Preserve all existing UI and features
